AMIMQTTClientJS
===============


Compiling AMIMQTTClientJS
=========================

* Requirements:

Make sure that [NodeJS](https://nodejs.org/) and [WebpackCLI](https://webpack.js.org/) are installed:
```bash
node --version
webpack --version
```

* Compiling:

```bash
npm install
webpack --mode production
```
